# PyDo — v3 Tkinter + SQLite

Interface graphique Tkinter avec base de données SQLite autonome.

> ⚠️ Version archivée — non maintenue. Le code actif basé sur cette version se trouve dans [`src/`](../../src/).

---

## Prérequis

- Python 3.x — [télécharger](https://www.python.org/downloads/) *(cocher "Add to PATH" lors de l'installation)*
- Modules Python :

```bash
pip install pillow tkcalendar
```

*(SQLite est inclus avec Python — aucune installation supplémentaire requise)*

---

## Lancement

```bash
python tout_doux_liste.pyw
```

La base de données est créée automatiquement au premier lancement.

---

## Accès à la base de données

Pour visualiser la base de données via une interface graphique, utiliser [DB Browser for SQLite](https://sqlitebrowser.org/).
