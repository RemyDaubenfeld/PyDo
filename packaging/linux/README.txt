PyDo - Installation Linux
==========================

Marche à suivre :

1. Ouvrez un terminal dans ce dossier
2. Rendez le script exécutable :
   chmod +x install.sh
3. Lancez l'installation :
   ./install.sh
4. PyDo apparaîtra dans votre menu d'applications.

Vous pouvez aussi lancer PyDo directement sans installation :
   chmod +x PyDo-linux
   ./PyDo-linux

Un fichier todolist.db sera créé automatiquement à côté de l'exécutable
pour stocker vos tâches.
